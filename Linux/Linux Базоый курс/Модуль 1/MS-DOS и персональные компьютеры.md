MS-DOS (Microsoft Disk Operating System) была выпущена в 1981 году для компьютеров IBM PC.

**Особенности архитектуры:**
* Написана для процессора 8086. Использовала **реальную адресацию памяти**: программы работали с ячейками памяти напрямую без контроля со стороны ОС.
* Отсутствие ядра: система была «вещью в себе» и позволяла любой программе обращаться к любому оборудованию напрямую.
* **Недостатки:** Программы могли мешать друг другу. Если две программы пытались использовать одну ячейку памяти, происходил сбой.

**Состав системы:**
* `IO.SYS` — расширение функций BIOS (базовой системы ввода-вывода).
* `MSDOS.SYS` — обработка прерываний (функций, упрощающих разработку).
* `COMMAND.COM` — командная оболочка для пользователя.
* Конфигурационные файлы: `CONFIG.SYS` (настройка системы) и `AUTOEXEC.BAT` (стартовый скрипт).

**Историческое значение:** Сделала ПО коммерческим продуктом. Была основой для Windows вплоть до версии Windows ME. Полный отказ от MS-DOS произошел только в Windows NT. Ограниченность MS-DOS (игнорирование возможностей процессора 386) подтолкнула Линуса Торвальдса к созданию Linux.
