### Файл: [[Структура каталогов Linux]]
В Linux структура каталогов является жесткой и иерархической. Все файлы и папки сводятся к единому корню — **/** (**root**). В отличие от Windows, система сама определяет, куда устанавливать ПО и где хранить настройки.

**Основные каталоги и их назначение:**
*   **/bin**: Основные исполняемые файлы и утилиты, необходимые для работы.
*   **/boot**: Информация для загрузки: загрузчик и само ядро системы.
*   **/dev**: Файлы устройств (так как в Linux всё есть файл).
*   **/etc**: Системные конфигурационные файлы и настройки служб.
*   **/home**: Домашние директории обычных пользователей.
*   **/root**: Домашний каталог администратора (Root). Вынесен отдельно, чтобы администратор мог работать, даже если диск с `/home` недоступен.
*   **/lib**: Библиотеки, необходимые для работы бинарных файлов.
*   **/media** и **/mnt**: Точки монтирования для съемных носителей (флешки, CD) и временного подключения дисков.
*   **/opt**: Место для установки дополнительных (optional) программных продуктов.
*   **/proc** и **/sys**: Виртуальные каталоги. `/proc` содержит информацию о процессах, а `/sys` — об устройствах, драйверах и параметрах ядра.
*   **/sbin**: Системные исполняемые файлы (system binary) для задач администрирования.
*   **/srv**: Данные сервисов (например, файлы FTP-сервера).
*   **/tmp**: Временные файлы.
*   **/usr**: Вторичная иерархия для данных пользователя, его программ и библиотек.
*   **/var**: Изменяемые данные, наиболее важен подкаталог **/var/log**, где хранятся журналы (логи) системы.
