**Процесс** — это работающая в данный момент программа, представляющая собой совокупность программного кода и ресурсов для его выполнения. 

**Механизм создания:**
В Linux новый процесс можно создать только из уже существующего.
1.  **Fork (Фок/Вилка):** Системный вызов, который делает полную копию родительского процесса. Вызов уникален тем, что **стартует в одном процессе, а завершается в двух** (в родителе и его клоне).
2.  **Exec (Экзек):** После клонирования системный вызов `exec` заменяет исполняемый код в копии на код нужной программы.
3.  **Exit (Экзит):** Завершение работы. Процесс сообщает **код возврата** (0 — успех, остальное — ошибка).
4.  **Wait (Вейт):** Процесс-родитель считывает код возврата своего потомка. Только после этого информация о потомке полностью удаляется из системы.
