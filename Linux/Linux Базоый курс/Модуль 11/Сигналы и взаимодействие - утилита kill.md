Поскольку [[Процессы-демоны|демоны]] не имеют терминала, взаимодействие с ними происходит через **сигналы**.

**Утилита kill:**
Синтаксис: `kill -[номер_сигнала] [PID]`.

**Основные сигналы (стандарт POSIX):**
1.  **SIGINT (2):** Прерывание (аналог `Ctrl+C`). Демонами не воспринимается с клавиатуры.
2.  **SIGTERM (15):** Сигнал завершения (по умолчанию). Просит программу завершиться корректно, освободив ресурсы. Процесс может игнорировать этот сигнал, если в нем написан соответствующий обработчик.
3.  **SIGKILL (9):** Безусловное завершение операционной системой. На него **невозможно написать обработчик** (кроме процесса Init).

**Исключения для SIGKILL (9):**
*   Процесс **Init** (игнорирует сигнал).
*   Процессы в статусе **D (Uninterruptible sleep)**, ожидающие ввода-вывода (не воспринимают сигналы).
