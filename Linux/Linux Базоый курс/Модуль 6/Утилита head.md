**Head** — утилита, которая выводит несколько первых строк файла («голову» файла).

**Параметры:**
* **По умолчанию:** Выводит первые **10 строк**.
* **Ключ `-n`:** Позволяет задать точное количество строк.
    * Можно писать кратко: `head -3 имя_файла`.
    * Рекомендуется писать полностью: `head -n 3 имя_файла`, чтобы избежать ошибок в будущих скриптах при использовании дополнительных ключей.
* **Применение:** Чаще всего используется для быстрого ознакомления со структурой текстового файла.
