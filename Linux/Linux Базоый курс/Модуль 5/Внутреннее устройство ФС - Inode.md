Файловая система (например, Ext4) использует блоки и иноды для хранения данных.

*   **Блок:** Минимальная единица хранения данных на диске (обычно 4 Кб). Файл в 1 байт все равно займет целый блок.
*   **Inode (Инода):** Индексный дескриптор, хранящий метаданные: права, владельца, GID, время создания и номера блоков с данными.
*   **Ограничения:** Количество инод фиксируется при создании файловой системы. Нельзя создать больше файлов, чем есть свободных инод, даже если на диске есть место.
*   **Команды:** `ls -i` (показать номер иноды), `stat` (вся информация из иноды), `df -i` (проверка свободных инод).
