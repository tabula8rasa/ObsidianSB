В Linux существует два типа ссылок, создаваемых утилитой `ln`.

**1. Hardlink (Жесткая ссылка):**
*   Это просто дополнительное имя для уже существующей иноды (синоним).
*   Файл существует, пока на его иноду указывает хотя бы один хардлинк.
*   **Ограничения:** Нельзя создавать для каталогов (кроме системных `.` и `..`) и между разными дисками (файловыми системами).

**2. Softlink (Символическая ссылка / `ln -s`):**
*   Аналог ярлыка Windows. Это отдельный файл, внутри которого записан путь к другому файлу.
*   **Преимущества:** Можно создавать для каталогов и между разными файловыми системами.
*   **Применение:** Удобно для обновления ПО (например, Java). Вместо изменения путей во всех программах, меняется только одна ссылка на новую версию.
