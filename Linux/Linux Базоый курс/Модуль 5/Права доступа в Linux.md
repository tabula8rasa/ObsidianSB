Для системы файл — это поток байт, но она должна знать права доступа на него.

**Типы прав:**
*   **r (read):** Чтение.
*   **w (write):** Запись.
*   **x (execute):** Исполнение.

**Категории доступа:**
1.  **Владелец** (User).
2.  **Группа владельца** (Group).
3.  **Все остальные** (Other).

**Формы записи прав:**
*   **Буквенная:** Например, `rwxr-xr-x`.
*   **Числовая (восьмеричная):** Основана на весах битов: **r=4, w=2, x=1**. 
    *   *Примеры:* 777 (все права всем), 644 (чтение/запись владельцу, остальным — только чтение).

**Управление:** Команда `chmod` позволяет менять права. Ключ `-R` применяет права рекурсивно к директории и всему её содержимому.
