Вся информация о пользователях и группах хранится в текстовых файлах в каталоге `/etc`.

**1. /etc/passwd**: Хранит основные данные (кроме паролей).
*   *Поля:* Логин, X (заглушка для пароля), UID, GID (основная группа), Комментарий, Домашний каталог, Командная оболочка (например, `/bin/bash`).

**2. /etc/group**: Хранит данные о группах.
*   *Поля:* Имя группы, X, GID, список участников через запятую.
*   **Утилиты:** `id` (вывод информации о пользователе) и `usermod` (модификация, например, добавление в группу через `usermod -aG`).

**3. /etc/shadow**: Содержит хэши паролей. Доступен только пользователю root.
