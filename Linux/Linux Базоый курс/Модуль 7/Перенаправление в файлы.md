Вывод любой команды можно перенаправить из терминала в файл с помощью специальных символов.

**Операторы перенаправления:**
*   **`>` (Один значок):** Перезаписывает файл. Если файл существовал, его содержимое удаляется, и записываются новые данные. Если файла не было, он будет создан.
*   **`>>` (Два значка):** Дописывает информацию в конец файла, сохраняя предыдущие данные.

**Работа с конкретными потоками:**
*   Символ `>` по умолчанию работает с потоком **STDOUT (1)**. Записи `> file` и `1> file` идентичны.
*   Для перенаправления ошибок используется **`2>`**.
*   Можно перенаправлять оба потока одновременно: `1> std_out 2> std_err`.
